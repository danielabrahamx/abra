# tech-stack.md: Physical Architecture (Abra MVP)

## 1. Frontend
- **HTMX (v1.9.10+):** For HATEOAS-compliant transitions.
- **Tailwind CSS:** Mobile-first utility classes.
- **Lucide-icons:** For action-oriented iconography (Maps, Delete, Add).

## 2. Backend & Data
- **Logic:** Node.js (Netlify Functions).
- **Storage:** `data/schedule.json`. 
- **Pattern:** Read-Modify-Write for JSON manipulation.
